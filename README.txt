-ЗАДАНИЕ НА ДАННУЮ ПРАКТИКУ ОТ РУКОВОДИТЕЛЯ ОРГАНИЗАЦИИ
  Кроссплатформенный клиент-сервер с плагинами. Сервер реализуется с помощью библиотек boost::beast или boost::asio и стандарта C++17. Клиент реализуется через браузер с помощью скриптов(JS), сервер получает пакеты данных Lau. У сервера существует база данных пользователей, у которых доступны/недоступны дополнительные возможности(плагины). Авторизация на сервер происходит по схеме Digest. Необходимо реализовать Rest API



-ОПИСАНИЕ ВЫБРАННОЙ ПРЕДМЕТНАЯ ОБЛАСТЬ И ЖЕЛАЕМОГО КОНЕЧНОГО ПРИЛОЖЕНИЯ
   Предметной областью является образовательная площадка для обучения, напоминающая сайт Studfiles, но с добавлением видеоуроков, аудио- и электронных книг. 
   Желаемое приложение будет использоваться для предоставления образовательных материалов пользователям. Приложение включает несколько уровней привилегий: обычные пользователи могут просматривать уже имеющуюся информацию, а администраторы имеют права на выкладывание новых материалов и редактирование уже существующих. В будущем для техподдержки будет создана отдельная сущность.
   Была разработана схема и примерный дизайн веб-страниц, находящихся в файле application_diagram.svg .
